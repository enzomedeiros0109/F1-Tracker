
# F1 Tracker

## O F1 Tracker é um projeto individual que mostra informações da última sessão de Fórmula 1, como a tabela do Campeonato de Pilotos e Construtores, tempos de volta, etc.

## Onde são obtidas as informações?

### O aplicativo coleta toda as informações de [openf1.org](https://openf1.org/), uma API gratuita de Fórmula 1.

## Como iniciar o programa?

### Clone o repositório usando git clone ou baixando o zip e extraindo a pasta
> git clone https://github.com/enzomedeiros0109/F1-Tracker.git

### Após clonar/extrair, digite no terminal
> cd F1-Tracker

### Rode no terminal
> py app.py
